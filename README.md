## 📖 Overview

Simon Says is a classic memory-based game that challenges players to remember and reproduce a sequence of colors generated by the computer. Each successful round adds a new color to the sequence, making the game progressively more difficult.

---

## ✨ Features

✅ Dynamic level progression

✅ Random color sequence generation

✅ Real-time answer validation

✅ Interactive visual feedback and animations

✅ Game-over detection with score display

✅ Easy restart functionality

✅ Clean and responsive user interface

---

## 🎯 How to Play

1. Press **any key** to start the game.
2. Watch the flashing color sequence carefully.
3. Repeat the sequence by clicking the colored buttons.
4. Complete the sequence correctly to advance.
5. Each level adds a new color to the pattern.
6. A wrong selection ends the game.
7. Press any key to start again.
